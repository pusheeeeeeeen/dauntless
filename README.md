# LiHv battery charger V1+V2
LiHv battery charger capable of charging LiHv batteries up to 4.35v, custom 3d designed case for 3d printing and an on-off switch. V1 is the simplest module, just to be available for cheaper and easier production of 4.35v LiHv chargers. V2 is equipped with a buzzer, OLED display, and XIAO RP2040 module to track battery percentage, and to alert the user when the battery is done charging.
